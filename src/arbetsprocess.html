Arbetsprocess



Började med att skriva npm init i en mapp och installerade parcel i sev-dev mode. Kopplade till Github och skapade en ny branch för utvecklingen.

Sedan startade jag npm start och skapade alla filer som index.html, css, javascript och kodade. Nästa steg lade jag upp bilderna och såg till att dessa optimerades automatisk till webP eller avif och komprimerade genom sharp.config.json.
