Uppgiften:



Uppgiften går på att skapa en demowebbplats som innehåller HTML, CSS och JavaScript. Där sidan skall innehålla en huvudmeny med stöd för ett godtyckligt antal menyalternativ. Sidan skall vara fullt responsiv.

Webbplatsen skall bestå av minst tre undersidor:
  • Startsida med info om uppgiften.
  • Undersida för bilder med minst fem stycken fotografier.
  • En undersida för publicering där du beskriver din skapade arbetsprocess.

Filerna skall vara väl strukturerade, med underkataloger för CSS, JavaScript och bilder. Det ska finnas skiljd filstruktur mellan arbetsfiler och publiceringsfiler.
